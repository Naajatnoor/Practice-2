# ⚽ FIFA World Cup 2026 Landing Page

A modern and responsive FIFA World Cup 2026 landing page built using **HTML5** and **CSS3**. This project was created as my **second frontend practice project** to strengthen my web development skills.

---

## 📖 About the Project

This project is inspired by the FIFA World Cup 2026 and focuses on creating a visually appealing and responsive landing page.

It includes a clean navigation bar, an engaging hero section, tournament statistics, host city information, match schedule, and a well-structured footer.

---

## ✨ Features

- ⚽ Modern Navigation Bar
- 🏆 Hero Banner Section
- 📊 Tournament Statistics
- 🌍 Host Cities Section
- 📅 Match Schedule Table
- 📱 Responsive Layout
- 🎨 Clean and Organized UI
- 💻 Built with Semantic HTML & CSS
